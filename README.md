Calculadora de Figuras Geométricas:
Aplicación que permita al usuario calcular el área y el perímetro de diferentes figuras geométricas (círculo, rectángulo y triángulo). 
La clase principal consta de varios menús para interactuar con el usuario.
Para la creación de figuras se le pide al usuario los datos que se pasan por parámetro a las clases correspondientes.
Captura errores como introducir texto en lugar de números para las opciones, mostrando un mensaje y permitiendo que se vuelva a 
introducir la opción valida sin interrumpir el programa. 
También captura errores si se introducen valores negativos en los valores de las figuras como su lado o base.
